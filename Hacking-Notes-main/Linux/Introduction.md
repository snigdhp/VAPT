### File Permissions

| Number | Permission Type       | Symbol |
|--------|-----------------------|--------|
| 0      | No Permission         | —      |
| 1      | Execute               | –x     |
| 2      | Write                 | -w-    |
| 3      | Execute + Write       | -wx    |
| 4      | Read                  | r–     |
| 5      | Read + Execute        | r-x    |
| 6      | Read +Write           | rw-    |
| 7      | Read + Write +Execute | rwx    |
